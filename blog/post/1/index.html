核心配置

Vlan batch 5 10
Int vlanif 5
Ip addr 126.67.0.254 24

Int vlanif 10
Ip addr 126.67.1.254 24

Int gi 0/0/1
Port link-type trunk
Port trunk allow-pass vlan 5 10

接入配置

Vlan batch 5 10

Int vlanif 5
Ip addr 126.67.0.1 24

Int gi 0/0/0
Port link-type trunk
Port trunk allow-pass vlan 5 10

Int gi 0/0/1
Port link-type acc
Port def vlan 10

Int gi 0/0/2
Port link-type acc
Port def vlan 10

Ip route-statci 0.0.0.0 0 126.67.0.254
